# Grunt/Backbone Broilerpane

Simple Broilerpan for Grunt/Backbone applications.

### Version

0.1.1

### Contains

* Backbone 1.2.3
* jQuery 2.2.0
* Lodash 4.5.0
* Require 2.1.9
* Moment 2.11.2
* Font-awesome: 4.5.0
* Ledger font-face

### Install

```sh
$ npm install -g grunt-cli
$ npm install
```

### Environment & commands

```sh
$ grunt development
$ grunt start
$ grunt production
```
